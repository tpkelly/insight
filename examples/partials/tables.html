Tables take an insight.Grouping and render the values in rows and columns. The table.columns() uses an array of objects with a label for the header and a value function.

This table uses world country data grouped by region. The table has been sorted using the descending() function. See the API docs to see what else you can do with tables.

var table = new insight.Table('stats', '#regions', worldCountriesByRegion) .columns([{ label: 'Total area (km2)', value: function(d) { return insight.formatters.numberFormatter(d.value.area.Sum); } }]) .descending(function(d) { return d.value.area.Sum; });
