The correlation coefficient represents how strong or weak a relationship is between two variables. A correlation of 0 means the parameters are unrelated, while a value closer to +1 or -1 means a very strong relationship.

Correlations can either be on raw data using insight.correlation.fromValues(xValues, yValues), or on a dataset using insight.correlation.fromDataSet(dataset, xFunction, yFunction). Both of these can also take an additional insight.ErrorContainer argument, which is populated with data about any errors encountered when processing the data.
